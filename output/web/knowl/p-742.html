More directly related to legislative analysis, proposed legislation is usually listed on state and territorial government webpages, but one would have to visit all of these 56+ pages, find a way of downloading the data into a spreadsheet, and continually monitor each of the pages for new bills in order to keep up with proposed legislation nationwide. Websites such as LegiScan 123  make this easier by aggregating data from all state legislatures, but in order to get this data into a form that can be imported into data analysis software, some knowledge of data scraping/scripting is required, and for large queries, paid API access may be necessary.
